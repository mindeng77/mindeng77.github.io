### docker 빌드

```shell
docker build -t 이미지이름 .
```

### 컨테이너 실행

```shell
docker run -d --name nginx -p 80:80 nginx:latest
```

### 컨테이너 접속

```shell
docker exec -it nginx /bin/bash
```

### 이미지 파일 저장

```shell
docker save -o nginx.tar nginx:latest
```

### 이미지 파일 로드

```shell
docker load < nginx.tar
```
