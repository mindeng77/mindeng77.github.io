인터넷이 되는 PC에서 플러그인 파일을 다운로드 하여 설치하는 방법

플러그인 파일은 vsix라는 확장자로 다음 사이트의 Visual Studio Code 카테고리에서 검색하여 다운로드 가능함.

<https://marketplace.visualstudio.com/>{:target="\_blank"}

1. VS Code 실행 후 Ctrl + Shift + P
1. install from VSIX 입력 후 선택
1. 다운로드 한 vsix 파일 선택
