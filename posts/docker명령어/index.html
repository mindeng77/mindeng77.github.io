docker명령어

By devobba
Posted 2023-04-25 Updated 2023-06-23 1 min read

docker 빌드

1

docker build -t {이미지이름:태그} .

컨테이너 실행

1

docker run -d --name {컨테이너이름} -p {호스트포트}:{컨테이너포트} {이미지이름:태그}

컨테이너 접속

1

docker exec -it {컨테이너이름} /bin/bash

이미지 파일 저장

1

docker save -o {파일명}.tar {이미지이름:태그}

이미지 파일 로드

1

docker load < {파일명}.tar
